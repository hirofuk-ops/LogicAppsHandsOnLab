# Step1. API Management インスタンスの作成
このセクションでは、Azure Portal を使用して API Management インスタンスを作成する手順を説明します。

1. Azure Portal にログイン
   - https://portal.azure.com にアクセス

2. 「リソースの作成」 → 「API Management」を選択

3. 基本設定
    - **名前**: apim-lab-demo（任意）
    - **リージョン**: Logic Apps と同じリージョン推奨
    - **SKU**: Consumption
    - **リソースグループ**: 既存または新規作成

4. 「確認と作成」 → 「作成」をクリック
