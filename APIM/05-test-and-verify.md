# API Management で Logic Apps API をテストおよび検証する手順
このセクションでは、API Management に追加した Logic Apps Standard の API をテストおよび検証する手順を説明します。

1. API Management インスタンスの「API」ページで、作成した Logic App API を選択

2. 「テスト」タブを開く

3. InvokeWorkflow 操作を選択

4. サブスクリプションキーが自動入力されていることを確認

5. Body に Logic Apps ワークフローに必要な JSON ペイロードを入力

6. 「送信」をクリック

7. レスポンスコード 200 OK と Logic Apps
